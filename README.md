## Running the program:
#### Prerequisites: 
* Node-js
* npm
* nodemon (optional, recommended)
#### Installation
1. Clone the repository to a destination of your choice. 
2. download AtomicCards.json from mtgjson and place it in the root directory of your project.
3. if you have nodemon  
    `nodemon index.js`  
    otherwise  
    `node index.js`
